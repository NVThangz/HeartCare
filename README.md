## HeartCare

1. Vào backend sửa file .env = notepad
```
MYSQL_USER=root
MYSQL_PASSWORD=root
```
thành
```
MYSQL_USER={Tên tài khoản mysql}
MYSQL_PASSWORD={Tên mật khẩu mysql}
```

2. Chạy file Tạo database.bat 1 lần
3. Chạy file Chạy server dev cổng 3000.bat để mở server dev
